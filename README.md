# economy-visualization
## 데이터로 시작하는 주식투자 내용정리

- 초기 퀀트 와 파생 상품 -> 블랙-숄즈 모형 등 파생상품 가격 결정 모델이 나옴 수학적으로  분석해 패턴을 찾는 알고리즘 매매

1) 멀티 팩터 전략 - 수악을 내는 특정 요인을 조합하여 투자하는 기법
   - 검증된 팩터를 조합하여 종목 포트폴이오를 직접 생성할 수 있는 점
   - 투자자 개인 성향이나 주식시장 환경에 따라 팩터 가중치 조절하여 유연한 결과 생성

 -> 감정적 편향 최소화 팩트 기반

### 대표적인 팩터
1) 모멘텀 - 주가나 거래량이 상승 추세이면 투자
2) 퀄리티(Quality) -  재무구조 우수, 매출과 이익 성장하는 기업
3) 밸류(Value) - 주가가 기업의 내재가치보다 저평가된 주식에 투자 ( 낮은 PER, 낮은 PBR)
4) 규모(size) - 시가 총액을 바탕으로
5) 변동성(Voaltility) - 주가 변동성 낮은 주식 투자
6) 배당(Dividend Yield) - 높은 배당 수익률 주식 투자

- 이 책에선 모멘텀 퀄리티 밸류 사용
### 모멘텀
- 주식 가격이나 거래량의 과거 추제가 미래에도 이어질것이는 가설기반
- 장점
  - 강한 추세상에서 높음 수익률
  - 시장심리 변화나 새로운 트렌드 파악 빠름
  - 데이터 수집이 쉽고 계산 용이

- 단점
  - 횡보장이나 변동성 장세에는 손실
  - 심리적인 요잉넹 의해 움직일수도
  - 버블에 편승하면 손실 입을수도

### 벨류
- 기업의 내재가치(자산,이익) 에 비해 주가가 저평가 된 종목을 찾는데 목적
- 주가 하락시 손실을 최소화할 수 있는 '안전마진' 확보 최우선
#### 장점
- 저평가된 우량 기업 찾아내어 주가의 하락 리스크 줄일 수 있고 높은 수익 기대
- 장기투자에 유리
- PER, PBR이 낮다는 것은 기업의 이익이나 자산 규모에 비해 주가가 싸다는 의미

#### 단점
- 지표만 낮은 함정주 일수도
- 미래성장성이 중요한 산업의 경우 PER,PBR 이 높게 나타나는 경우가 있음
- 분기실적이 발표되는 시점마다 업데이트 되기 때문에 즉시성이 떨어짐

### 퀄리티
- 기업의 이익창출 능력, 재무건전성, 사어모델 우수성 등등 평가
- 좋은 기업은 이미 시장에서 평가가 높아 비쌀수도 있


-> 팩터를 융합 활용하면 하나의 팩터가 갖는 단점을 보완

### 멀티팩터 전략 유의 사항
1) 숫자 너머의 진실 파악 뉴스나 리포트를 통해 최신 트렌드 모니터링
2) 손절가격을 사전에 설정하여 감정적인 대응 배제
3) 서로 다른 성격의 팩터를 조화롭게 구성하여 전략간의 상관관계 낮추는 것이 중요
